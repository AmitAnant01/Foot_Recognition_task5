## Food Recognition and Calorie Estimation

This project is designed to identify different food items from images and estimate their calorie content. It helps users track their dietary intake and make informed food choices.

## Dataset

This project uses the [Food-101 dataset](https://www.kaggle.com/dansbecker/food-101) from Kaggle. It consists of **101,000 images** across **101 food classes**, with 750 training and 250 test images per class.

## Tech Stack
- PyTorch / TensorFlow
- OpenCV, PIL
- Pandas, NumPy
- Matplotlib, Seaborn
